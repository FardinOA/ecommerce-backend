# Install Dependencies

**For Backend** - `npm i`

**For Frontend** - `cd frontend` ` yarn`

## Env Variables

Make Sure to Create a .env file in backend/.env directory and add appropriate variables in order to use the app.

**Essential Variables**
PORT=

STRIPE_API_KEY=
STRIPE_SECRET_KEY=
JWT_SECRET=

COOKIE_EXPIRE=
SMPT_SERVICE =
SMPT_MAIL=
SMPT_PASSWORD=
SMPT_HOST=
CLOUDINARY_NAME
CLOUDINARY_API_KEY
CLOUDINARY_API_SECRET
